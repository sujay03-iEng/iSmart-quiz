NEPLAN – Power System Analysis

NEPLAN Software-360 is a complete browser-based power system analysis tool in the market, and it offers all advantages of cloud computing. The software does not require installation on a specific desktop or laptop—it can be accessed from anywhere via the internet. Hence, new licensing models are available for the users.

The software is designed for flexible deployment, allowing use either in the cloud via the Internet or within a customer’s intranet, depending on their preferred hosting environment. This flexibility ensures that network projects can be securely stored in one of three formats: a multi-user SQL database in the cloud, a multi-user SQL database within the customer’s intranet, or a local XML file.

NEPLAN | Electricity

NEPLAN Electricity is a software tool used for analyzing, planning, optimizing, and simulating electrical networks. It offers a user-friendly interface and a modular concept covering transmission, distribution, and generation networks. It is ideal for renewable energy systems and SmartGrid applications, providing high-accuracy simulations, including wind and solar powered plants. NEPLAN integrates with Matlab/Simulink, offering both desktop and web/cloud solutions, with a common database and API for easy external integration.

NEPLAN | Gas / Water / Heating

NEPLAN Gas, Water, and Heating is a software tool used for analyzing, planning, optimizing, and simulating networks. Its user-friendly interface and modular concept support efficient study case performance. NEPLAN uniquely allows simultaneous simulation of electricity, gas, water, and heating networks, enabling users to model their interactions. Available as a desktop application, web/cloud solution, or intranet service, NEPLAN offers flexible technological options for various network simulations and calculations.

NEPLAN | Asset Management

The NEPLAN Asset Management Tool answers the questions “what happens if…”. This tool offers outmost flexibility modelling any system. It allows the asset managers to evaluate each asset individually and define a medium and long-term asset strategy in order to optimize the investment and maintenance planning as well as power supply reliability. The customizable software has a modular concept and covers all aspects of Operational Asset Management Strategic Asset Management. The software is available across multiple platforms, including a desktop application, a cloud-based web browser solution, an intranet deployment, and as a web service.
